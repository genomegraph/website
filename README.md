## Genome Graph 研究会

### グラフゲノムとは

ゲノムの情報を従来の線形な表現ではなく、数学的なグラフ理論に基づいて表現し、解析を行う概念のことです。
アセンブリやSV検出、メタゲノム解析などで応用できると期待されています。

![例：SNPのグラフ表現](images/SNP_IMAGE.png)
![例：ちょっと複雑な構造のグラフ表現](images/GraphGenomeImage.png)
構造多型（SV）は以下のようにグラフで表現できます。
